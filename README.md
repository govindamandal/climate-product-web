# Climate Product Web

React frontend for a B2B climate product management platform. It includes product lifecycle management, Digital Product Passports, sustainability analytics, AI advisor flows, toast feedback, protected routes, and product image upload UX.

## Stack

- React 19
- TypeScript
- Vite
- TanStack Query
- React Router
- Zustand
- TailwindCSS
- Recharts
- Vitest

## Local Setup

```bash
pnpm install
pnpm dev
```

Create `.env` when pointing to a non-default API:

```bash
VITE_API_URL=http://localhost:8000/api/v1
```

## Validation

```bash
pnpm lint
pnpm test
pnpm build
```
